UUID - Universally Unique Identifier

это 128-битный уникальный идентификатор, который записывается в виде строки из 36 символов

Допустимы шестнадцатеричные символы: цифры 0-9 и буквы a,b,c,d,e,f

Формат:  xxxxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx

Ex: 550e8400-e29b-41d4-a716-4466554400000

+ : УНИКАЛЬНОСТЬ - рандомная генерация (создание случайного значения)

вероятность коллиззии (повторения значения)

Возможное количество уникальных значений: 2^128 (~ 3 * 10^38)

При создании 1 миллиарда значений вероятность повтора 10^(-18)
